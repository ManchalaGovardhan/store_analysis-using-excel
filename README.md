E-Commerce Sales Dashboard Overview
Project Description
This project presents a high-level overview of e-commerce sales performance using key metrics such as order volume, sales amount, customer demographics, order status, and channel distribution. The data is intended for quick analysis of business performance and customer behavior.

Insights
Sales Performance: Total orders (2.00M) slightly exceed total sales amount (1.90M), indicating potential discounts or low-value orders.

Gender Distribution: Orders are fairly evenly split between Men (1.70M) and Women (1.60M).

Order Status: Delivered orders (45,844) significantly outnumber canceled ones (5,110), suggesting high fulfillment efficiency.

Channel Concentration: 100% of orders come from a single channel, which may represent a risk if the channel faces issues.

Age Segment: 40% of orders are attributed to a specific age group — useful for targeted marketing.
